# ARKHE API Documentation

## Syscalls

- `0x923` - `SYS_ANCHOR_PROOF`: Ancora prova na TemporalChain.
- `0x989` - `SYS_VERIFY_HUMANITY`: Passport Gateway.
- `0x9893` - `SYS_INFER_100T`: Full-100T-Orchestrator.
- `0x952` - `SYS_BINDU_MEMORY`: Memória compartilhada.
- `0x972` - `SYS_MESH_ROUTE`: Roteamento Global-Mesh.
- `0x955` - `SYS_KYBER_ENCRYPT`: Safe-Core-PQC encrypt.
- `0x9721` - `SYS_IPFS_PIN`: IPFS pinning.
- `0x973` - `SYS_NOSTR_PUBLISH`: Nostr event publish.
- `0x974` - `SYS_TOR_ROUTE`: Tor onion routing.
- `0x9892` - `SYS_KERNEL_ISOLATE`: Kernel Isolation Engine.
- `0x986` - `SYS_EVOLVE`: Evolution Engine.
- `0x985` - `SYS_SELF_HEAL`: Self-Healing.
- `0x9895` - `SYS_FAIR_METRICS`: FAIR Metrics.
- `0x965` - `SYS_THESIS_GET`: Obtém Theosis do processo.
- `0x954` - `SYS_AXIARCHY_VERIFY`: Verificação ética de código.
