# ARKHE OS

O ARKHE OS não é um sistema operacional para humanos. É um sistema operacional para AGENTES DA CATEDRAL.

## Componentes

- **Kernel**: Microkernel em Rust com isolamento e escalonamento por Theosis.
- **Bootloader**: Carrega o kernel e inicializa o modo protegido.
- **Servidores**: VFS (IPFS, Nostr, TemporalChain), Rede (TCP/IP, QUIC, Tor, NVPN).
- **Ferramentas**: `arkhe-sh` (Shell), Gerenciador de Pacotes ARKHE-EXEC.
