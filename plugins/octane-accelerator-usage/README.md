# Octane Accelerator Usage Plugin

This plugin provides the `accelerator-usage` skill to understand how the Octane Accelerator integrates with the Octra wallet extension, how to use its HTTP/WebSocket API, run benchmarks, and configure remote proving via encrypted relay tunnels.

## Structure

- **`.claude-plugin/plugin.json`**: Plugin metadata.
- **`skills/accelerator-usage/SKILL.md`**: Model-invoked skill containing the documentation on how the Octane Accelerator works.
